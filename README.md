# Nostra - E-commerce Website

Nostra is a responsive e-commerce fashion website built using **HTML**, **TailwindCSS**, **CSS**, and **JavaScript**.  
The project includes a Home page, Collections page, and Contact Us page with working search and filter functionality.

## Live Reference

This project was inspired by the Nostra e-commerce website reference:  
https://errormakesclever.github.io/Nostra-Ecommerce-Js/index.html

## Pages Included

- `index.html` - Home Page
- `collections.html` - Collections Page
- `contact.html` - Contact Us Page

## Features

- Fully responsive website layout
- Modern e-commerce homepage design
- Collections page with product cards
- Search functionality using JavaScript
- Filter functionality using JavaScript
- Contact Us page with contact form layout
- Mobile-friendly navigation menu
- Different product and fashion images used

## Technologies Used

- HTML5
- TailwindCSS
- CSS3
- JavaScript
